# Data Structures

This page documents the data structures in FAR.

***

## Alarm System

```lua
alarmSystem.id
alarmSystem.name
alarmSystem.status
alarmSystem.lastStatusChange
alarmSystem.passcode
alarmSystem.location
alarmSystem.aop
alarmSystem.monitoring
alarmSystem.customdata
alarmSystem.postalCode
alarmSystem.position.x
alarmSystem.position.y
alarmSystem.position.z
```

<Tabs>
	<TabItem value="alarmSystem.id" label="id" default>
		<p>`int`</p>
		<p>ID of the Alarm System.</p>
		<p>Example: `1`</p>
	</TabItem>

	<TabItem value="alarmSystem.name" label="name">
		<p>`string`</p>
		<p>Name of the Alarm System.</p>
		<p>Example: `Mission Row Police Station`</p>
	</TabItem>

	<TabItem value="alarmSystem.status" label="status">
		<p>`enum`</p>
		<p>Status of the Alarm System as an enum.</p>
		<p>Values:</p>
		- `0` - Invalid status
		- `1` - Entire system disabled
		- `2` - System ready / standby
		- `3` - Announcement in progress
		- `4` - WalkTest in progress
		- `5` - System has isolations
		- `6` - System in Alarm
		- `7` - System silenced
		- `8` - System test in progress

		<p>Example: `2`</p>
	</TabItem>

	<TabItem value="alarmSystem.lastStatusChange" label="lastStatusChange">
		<p>`date`</p>
		<p>The last time the Alarm System's status was changed.</p>
		<p>Example: `2024-12-23T04:14:33.7711836Z`</p>
	</TabItem>

	<TabItem value="alarmSystem.passcode" label="passcode">
		<p>`string`</p>
		<p>Passcode for Control Panels in this Alarm System.</p>
		<p>Example: `1234`</p>
	</TabItem>

	<TabItem value="alarmSystem.location" label="location">
		<p>`string`</p>
		<p>Human readable location of this Alarm System.</p>
		<p>Example: `Altee St, Mission Row`</p>
	</TabItem>

	<TabItem value="alarmSystem.aop" label="aop">
		<p>`string` | `null`</p>
		<p>Optional AOP used for Random Alarm Activations.</p>
		<p>Example: `Los Santos`</p>
	</TabItem>

	<TabItem value="alarmSystem.monitoring" label="monitoring">
		<p>`bool`</p>
		<p>A boolean indicating if Monitoring for this Alarm System is active or inactive.</p>
		<p>Example: `true`</p>
	</TabItem>

	<TabItem value="alarmSystem.customdata" label="customdata">
		<p>`string` | `null`</p>
		<p>Optional Custom Data for this system. <a href="./tool#adding-custom-data">See here</a> for more info.</p>
		<p>Example: `Your Custom Data Goes Here!`</p>
	</TabItem>

	<TabItem value="alarmSystem.postalCode" label="postalCode">
		<p>`string` | `null`</p>
		<p>Postal Code this Alarm System is located within.</p>
		<p>Example: `123`</p>
	</TabItem>

	<TabItem value="alarmSystem.postion" label="position">
		<p>`table`</p>
		<p>World coordinates of the center of this Alarm System.</p>
		<p>Example:</p>
		```lua
		local position = vector3(alarmSystem.position.x, alarmSystem.position.y, alarmSystem.position.z)
		```
	</TabItem>

</Tabs>
