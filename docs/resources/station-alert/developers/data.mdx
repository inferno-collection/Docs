# Data Structures

This page documents the data structures in SA.

***

## Station Location

```lua
location.name
location.group
location.customdata
location.position.x
location.position.y
location.position.z
location.doors
```

<Tabs>
	<TabItem value="location.name" label="name">
		<p>`string`</p>
		<p>Name of the Location.</p>
		<p>Example: `Station One`</p>
	</TabItem>

	<TabItem value="location.group" label="group">
		<p>`string` | `null`</p>
		<p>Optional group this Station belongs to.</p>
		<p>Example: `Los Santos`</p>
	</TabItem>

	<TabItem value="location.customdata" label="customdata">
		<p>`string` | `null`</p>
		<p>Optional Custom Data for this location. <a href="./tool#adding-custom-data">See here</a> for more info.</p>
		<p>Example: `Your Custom Data Goes Here!`</p>
	</TabItem>

	<TabItem value="location.postion" label="position">
		<p>`table`</p>
		<p>World coordinates of this Station Location.</p>
		<p>Example:</p>
		```lua
		local position = vector3(location.position.x, location.position.y, location.position.z)
		```
	</TabItem>

	<TabItem value="location.doors" label="doors">
		<p>`table`</p>
		<p>A table containing all the door names of this Station Location.</p>
	</TabItem>
</Tabs>

## Alert

```lua
alert.tone
alert.tones
alert.message
alert.locations
alert.unitcolors
alert.customdata
```

<Tabs>
	<TabItem value="alert.tone" label="tone">
		<p>`string|null`</p>
		<p>The tone that played for this Alert.</p>
		<p>If `null`, use `tones`.</p>
		<p>Example:</p>
		```lua
		Tone Two
		```
	</TabItem>

	<TabItem value="alert.tones" label="tones">
		<p>`table`</p>
		<p>The tones that played for each station in this Alert.</p>
		<p>If empty, use `tone`.</p>
	</TabItem>

	<TabItem value="alert.message" label="message">
		<p>`string|null`</p>
		<p>The TTS message for this Alert.</p>
		<p>Example:</p>
		```lua
		Station 2, Engine 2, Fire Call!
		```
	</TabItem>

	<TabItem value="alert.locations" label="locations">
		<p>`table`</p>
		<p>A table containing the Location Names for this Alert.</p>
	</TabItem>

	<TabItem value="alert.unitcolors" label="unitcolors">
		<p>`table`</p>
		<p>A table containing the unit colors this Alert.</p>
	</TabItem>

	<TabItem value="alert.customdata" label="customdata">
		<p>`any`</p>
		<p>Any data passed here will be send to the [Call Screen](../config#custom-alert-url).</p>
		<p>Data will be encoded as JSON and then sent as a URI Encoded string.</p>
	</TabItem>
</Tabs>
