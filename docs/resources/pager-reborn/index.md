## Available Versions
There are two versions of Pager Reborn.

### Pager Reborn
The standalone version of Pager Reborn is just called "Pager Reborn" or "Pager Reborn (Standalone)".  
It supports all features, including full Sonoran CAD integration (even with with the Free version of Sonoran CAD).

Pager Reborn can be purchased [here](https://store.inferno-collection.com/resources?modal=7367879).

### Pager Reborn - Sonoran Edition
Sonoran Edition is a lite version of Pager Reborn that is provided to Sonoran CAD Pro customers for free.  
Some important notes for Sonoran Edition:

- Only works for the lifetime of the subscription.
  - Downgrading or canceling will result in Sonoran Edition ceasing to function.
- Only Sonoran CAD integration works.
  - All other activation types (UI, exports, third-party resources, etc.) will not work.

Pager Reborn - Sonoran Edition can be downloaded [here](https://store.inferno-collection.com/sonoran).
