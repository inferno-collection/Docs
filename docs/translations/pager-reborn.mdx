Below you can find several translations for PR. The language files are provided with the resource, inside the `languages` folder, but translated `sounds.json` can be found on this page.

## `sounds.json`
In the root directory of the resource is a file called `sounds.json`. This file contains the names and file paths of all the alert tones and vibrations used by the resource. The names of the sounds can be changed by server owners or developers to be anything in any language, however, if you would like to use the default names but in your own language, you can find their translations below.
